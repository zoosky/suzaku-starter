# Suzaku starter
A starter project for using the Suzaku framework.

1. `git clone https://github.com/suzaku-io/suzaku.git`
1. `cd suzaku`
1. `sbt publishLocal`
1. `cd ..`
1. `git clone https://github.com/suzaku-io/suzaku-starter.git`
1. `cd suzaku-starter`
1. `sbt`
1. \> `re-start`
1. Open browser on `http://0.0.0.0:12121`.
